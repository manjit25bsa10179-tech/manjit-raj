
Bank Management System
Overview
This is a simple console-based Bank Management System implemented in Python. It allows users to create bank accounts, deposit money, withdraw money, and check account balances. The project demonstrates core programming concepts such as object-oriented programming (OOP), functions, and user input handling.

Features
Create new bank accounts with account number, account holder name, and initial deposit.

Deposit money into an existing account.

Withdraw money from an account with balance validation.

Check account balance and account holder details.

Menu-driven interface for ease of use.

Requirements
Python 3.x

How to Run
Clone or download the project files.

Open a terminal or command prompt.

Navigate to the project directory.

Run the program with: python bank_management.py

Follow the on-screen menu options.

Usage
Select options from the menu to perform banking operations.

Enter the required information such as account number, amount, and account holder name when prompted.

Use option 5 to exit the program.

Code Structure
Account class: Manages individual account details and operations.

Bank class: Maintains multiple accounts and handles banking functionalities.

main function: Drives the menu and user interaction loop.

Future Enhancements
